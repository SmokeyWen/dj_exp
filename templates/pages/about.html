Painkey, Paint with Smokey

I might have discovered what framework I like the most.


Painkey is a Django project that I decided to work on while staying at home and searching for a job. Due to the pandemic, job searching was difficult for people like me who graduated from colleges before it happened. In order to keep myself busy and hopefully pick up a few new skills, I started to read through Django documents and online tutorials.

I really like to work with Python, since it is easy to read and understand. After a quick refresh on Python syntax, I started this Django project. Django gives me a brand new way of thinking on planning and designing the structure of a web application. As I was working on this project, new ideas kept popping up in my mind and I will continuously work on Painkey to make it more presentable.


What Happens Next?

There are still flaws in this web application, on both front end and back end. I will take my time to fix them and hopefully refactor and organize its file system to make it easier for me to add more features in the future. Below are some of the ones already on my mind:

  • Comment section below each miniature
  • Like counter for each miniature page
  • To-do list for both this website and the miniatures I own but haven't painted

Club Managers and atheletes can view each other's profiles depending on their identities and like or dislike by swiping right and left. Parties are matched when they like each other.
